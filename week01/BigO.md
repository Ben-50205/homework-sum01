## Big-O 分析：f(n) = 3n³ + n² + n

### 步驟一：分析每一項的成長速度

- `3n³`：立方項，**成長最快**
- `n²`：平方項
- `n`：線性項

當 `n → ∞`（n 趨近無限大）時，**高次項主導整體成長速度**。在這裡，`3n³` 是主導項。

---

### 步驟二：根據定義找出上界

根據 Big-O 定義：

f(n) = 3n³ + n² + n ≤ c · n³，對所有 n ≥ n₀
